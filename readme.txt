This project is a responsive, interactive live poster web application that visually represents the four quadrants of Applied Computing at Purdue CIT: Software Development, Cybersecurity, Networking, and Data Management. The application aims to provide prospective students and employers with career pathways, job descriptions, and academic suggestions for each area. A quiz helps users discover which concentration aligns with their interests.

Features
Four Quadrants: Interactive quadrants representing each concentration, with modals containing career descriptions, degree pathways, and sample job titles.

Quiz: A 10-question quiz that helps users find the best-fit concentration.

Responsive Layout: Designed with Bootstrap's grid system and custom media queries for optimal viewing across mobile, tablet, and desktop screens.

Custom Favicon: A gear-shaped favicon with four color-coded quadrants representing each CIT area.

Accessibility: Semantic HTML5, ARIA attributes, and proper contrast for accessibility compliance.
